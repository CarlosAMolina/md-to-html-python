# md-to-html-python

## Introduction

Project to convert md files to html.

This project is forked from https://github.com/CarlosAMolina/cmoli.es-deploy.

This project uses Go to coordinate actions, but the conversion logic is written in Python.

### Required software

- [Docker](https://www.docker.com/). You need to have Docker installed and run it as [rootless](https://docs.docker.com/engine/security/rootless/).
- [Git](https://git-scm.com/).
- [Go](https://go.dev/).
